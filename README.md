# GenDiff

## Description

This is a training project 2 for Hexlet PHP developer course.
The utility compares 2 plain json/xaml files.

## Requirements

* php 8.3.0+
* composer

## Install

```sh
git clone https://github.com/eviktor/php-project-48.git
cd php-project-48
make install
```
